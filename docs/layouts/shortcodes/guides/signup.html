The first step to deploying anything on Platform.sh is, of course, to sign up for an account. If you do not already have a {{ .Get "name" }} site to deploy, and you are going to follow along with this guide using the {{ .Get "name" }} template, you can click the "Deploy on Platform.sh" button below to sign up for an account and initialize your project all in the same step.

If you are deploying your own pre-existing {{ .Get "name" }} site, you will need to visit the Platform.sh accounts page and fill out your information to set up your trial account.

Note:

For both cases, you can sign up using an existing GitHub, Bitbucket, or Google account. If you choose this option, you will be able to set a password for your Platform.sh account later.
